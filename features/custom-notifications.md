## Custom Notifications

Custom notifications in eLMS allow users to create and send personalized messages for all users, specific user, students, parents, and teachers which will shown on notification section in applications. These notifications can be created only by admin to specific events, such as important announcements, upcoming school activities, academic updates, and more. With the flexibility to choose recipients, message title, message content.
